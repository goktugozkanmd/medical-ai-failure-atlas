# Clinical AI Literacy Simulator Module

Date: 2026 07 08

Status: local education module. Not externally released.

Purpose: turn MedFailBench failure cases into a short clinician AI literacy simulator without patient data or clinical deployment claims.

## Twenty Minute Flow

1. Show one synthetic prompt.
2. Show an unsafe or incomplete model answer.
3. Ask the reviewer to identify the missing danger signal.
4. Show the safer answer pattern.
5. Run SourceCheckup style source claim review when the answer cites evidence.
6. Close with what cannot be claimed from the exercise.

## Local Inputs

1. Existing synthetic MedFailBench prompts.
2. Existing public safe failure cards.
3. Existing SourceCheckup source support examples.
4. SafetyGuard Studio for live scoring.

## Boundary

No patient data, no medical advice, no clinical validation, no institution claim, no official collaboration claim, and no external presentation without owner approval.
