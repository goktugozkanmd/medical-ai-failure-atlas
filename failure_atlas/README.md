# Failure Atlas

The public Failure Atlas materials live in `failure_atlas/public/`.

## Public Files

| Path | Purpose |
| --- | --- |
| `public/INDEX.md` | Public entry point for the atlas. |
| `public/METHODOLOGY.md` | Method summary and boundaries. |
| `public/TAXONOMY_PREVIEW_20260616.md` | Failure taxonomy preview. |
| `public/CASE_INTAKE_CHECKLIST_V0_1.md` | Case intake checklist. |
| `public/case_intake_schema_v0_1.json` | Intake schema. |
| `public/case_intake_examples_v0_1.jsonl` | Synthetic examples. |
| `public/clinician_review_states_v0_1.json` | Review state definitions. |
| `public/taxonomy_map_v0_1.json` | Taxonomy map. |
| `public/build/` | Generated public summaries. |

Use synthetic or public examples only. Do not add patient data or private clinical text.
